# BMI 계산기

키와 몸무게를 입력하면 BMI(체질량지수)를 계산하고, 체중 상태를 분류한 뒤 상태에 맞는 건강 관리 조언을 보여주는 프로그램입니다. Python 스크립트(CLI)와 이를 활용한 웹앱(Flask) 두 가지 방식으로 사용할 수 있습니다. 이 폴더만으로 완전히 독립적으로 동작합니다.

## 파일 구성

```
BMI_Calculator/
├── bmi_calculator.py     # BMI 계산 및 분류 로직 (CLI 실행 가능, 웹앱에서도 import해서 재사용)
├── bmi_app.py             # BMI 웹사이트 (Flask, 포트 5000)
├── templates/
│   └── bmi.html          # BMI 입력 폼 + 결과 화면
├── static/
│   └── style.css         # 그라데이션 없는 플랫 디자인 스타일시트
├── requirements.txt       # 필요한 패키지 목록
└── README.md
```

## 기능

1. 키(cm)와 몸무게(kg)를 입력받아 BMI 계산 (`체중(kg) / 키(m)^2`)
2. BMI 값을 기준으로 5단계 상태 분류
3. 각 상태별로 건강 관리 조언 4가지를 순서대로 제공
4. 숫자가 아니거나 0 이하의 값을 입력하면 오류 안내

### BMI 분류 기준 (대한비만학회 기준)

| BMI 범위 | 상태 |
|---|---|
| 18.5 미만 | 저체중 |
| 18.5 ~ 23 미만 | 정상 |
| 23 ~ 25 미만 | 과체중 |
| 25 ~ 30 미만 | 비만 |
| 30 이상 | 고도비만 |

## 사전 준비

```
pip install -r requirements.txt
```

(Flask만 설치됩니다. API 키가 필요 없습니다.)

## 실행 방법

### 1. CLI 버전 (터미널에서 직접 실행)

```
python bmi_calculator.py
```

터미널에 키와 몸무게를 입력하면 결과와 조언이 바로 출력됩니다.

### 2. 웹앱 버전 (브라우저에서 실행)

```
python bmi_app.py
```

- 서버가 실행되면 1초 후 기본 브라우저가 자동으로 `http://127.0.0.1:5000` 을 엽니다.
- 자동으로 열리지 않으면 브라우저에서 직접 주소를 입력해서 접속합니다.
- 종료할 때는 터미널에서 `Ctrl + C`를 누릅니다.

## 작동 방식

- `bmi_app.py`는 `bmi_calculator.py`의 BMI 계산/분류/조언 함수를 그대로 import해서 사용합니다.
- 이 폴더는 다른 폴더(날씨/대기질 사이트 등)를 참조하지 않는 완전히 독립된 프로젝트입니다. 폴더 전체를 다른 위치로 옮기거나 별도 저장소로 분리해도 그대로 동작합니다.
